\# ex02 ESP32 millis无阻塞LED闪烁实验

\## 1. 实验原理

1\. millis()函数返回设备上电总毫秒数，类型为unsigned long，无阻塞不占用程序运行；

2\. 通过时间差值判断是否到达1秒，翻转LED电平，替代阻塞函数delay()；

3\. 优势：循环内可同时运行其他任务，不会被延时卡住。



\## 2. 硬件配置

\- 主控：ESP32 Dev Module

\- LED引脚：GPIO2（开发板自带板载LED）

\- 串口波特率：115200



\## 3. 运行现象

LED以1Hz频率循环亮1秒、灭1秒；串口监视器每秒打印当前运行毫秒与LED状态。



\## 4. 调试注意

1\. 串口无输出：检查USB数据线是否支持数据传输、正确选择COM端口；

2\. LED不闪烁：确认开发板型号选择ESP32 Dev Module，引脚定义为GPIO2。
